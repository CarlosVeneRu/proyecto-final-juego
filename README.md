# Proyecto Final - Juego Gato


## Descripción del Juego

Este proyecto es sobre el clásico juego de Gato. El usuario juega contra una la máquina que puede responder con movimientos aleatorios. Se permite elegir símbolo que en este caso será X u O, se alternan los turnos, y el sistema detecta automáticamente al ganador o un empate.

## Objetivo 

Desarrollar el juego de “Gato” en el que el jugador compite contra la máquina. El objetivo es aplicar conceptos de lógica, programación estructurada que simule decisiones inteligentes por parte de la máquina, al mismo tiempo que se trabaja colaborativamente bajo una metodología ágil que en este caso será Kanban.

## Tecnologías Utilizadas

- Lenguaje: Python  
- Control de versiones: Git y GitHub  
- Gestión de tareas: Trello
